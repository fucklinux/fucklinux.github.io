edit 介绍

Fuck Linux 操作系统：一个被放弃的项目（本来就没打算做），最初由“你来坑我啊”提出。由于群内没有人会做内核，我写了一个奇怪的脚本，用来替代它。

library_books 文档

这里记录了 Fuck Linux 的使用方法。

1.首先，你需要准备 Linux 环境。如果已经准备好，请转到第 2 步。
如果你在使用 Android 设备，可以使用 ZeroTermux 来模拟Linux环境。

2.在这里或点击上面的“立即下载”按钮获取 Fuck Linux 安装器（也可以执行wget https://fucklinux.github.io/installer/install.sh），并在安装器目录依次执行以下指令：
chmod +x install.sh
./install.sh
1.5.0 版本已经有了图形界面，安装时按照提示操作即可。
你可以在以后执行 ./start-fuck.sh来启动 Fuck Linux 。
